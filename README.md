# 竞赛作品主题奥卡姆蒸馏法

> Competition Theme Occam Distillation — 从 N 个参赛项目中蒸馏 ≤5 个核心方向的系统方法

---

## 这是什么

一套分析竞赛/黑客马拉松/大赛作品的**思维框架**，三步骤从散乱项目中提炼核心主题和可执行行动指南。

不是翻译项目介绍，而是**蒸馏本质共性**。

## 核心方法

```
扫读归因（≤8字标签）
    ↓
聚类归并（按"解决谁的问题"分组，≤4项/组）
    ↓
行动映射（本周 → 本月 → 长期）
```

## 适用场景

- 🏆 黑客马拉松 / 创业大赛作品评审
- 📊 竞品分析（多产品横向对比）
- 🔭 技术趋势扫描
- 💡 团队灵感板（作品→改造清单）

## 文件结构

| 文件 | 内容 |
|------|------|
| `SKILL.md` | 核心方法论 + 输出模板 |
| `QUICKREF.md` | 三步法速查 + 检查清单 |
| `TEST.md` | 测试案例（1真实+1虚构） |
