# 竞赛作品主题蒸馏 — 快速参考

## 三步法速查

```
扫读归因 → 聚类归并 → 行动映射
(≤8字)    (≤4项/组)   (本周→本月→长期)
```

## ≤8字法则示例

| 原描述 | 蒸馏后 |
|--------|--------|
| "自进化评估驱动的AI Agent团队，在编码工具中运行" | 多Agent对抗协作 |
| "AI原生数字员工，有身份档案和主动感知能力" | AI当同事 |
| "多智能体编排的投研分析平台" | 多Agent交叉验证 |
| "基于意图绑定的技能权限治理框架" | 技能权限审计 |

## 聚类规则

| 按什么分 | ✅/❌ |
|----------|-------|
| 解决谁的问题 | ✅ |
| 核心技术/框架 | ❌ |
| 交互方式(Web/CLI) | ❌ |
| 团队背景/规模 | ❌ |

## 输出检查清单

- [ ] 每个项目有 ≤8字 本质标签
- [ ] 核心主题 ≤5 个
- [ ] 每个主题下项目 2-4 个
- [ ] 分点总结：关键词 + 核心 + 空行分隔
- [ ] 行动指南：本周/本月/长期 三层
- [ ] 最终有"一句话总结"

## 一句话总结公式

```
N个项目 → X个方向 → 1个终极问题 → "如何[核心矛盾]"
```
